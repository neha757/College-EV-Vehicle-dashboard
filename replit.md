# EV Requisition Management System

## Overview
A Flask web application for managing electric vehicle requisitions in a college environment with role-based access control and status tracking functionality.

## Created
October 30, 2025

## Recent Changes
- Initial project setup with Flask framework
- Configured SQLite database for user and requisition management
- Implemented session-based authentication with role-based access control

## Project Architecture
- **Backend**: Flask 3.0.0 with SQLite database
- **Frontend**: Bootstrap 5 with custom orange color theme
- **Authentication**: Session-based with role management (Admin vs Student/Faculty)
- **Database**: SQLite with tables for users and requisitions

## Core Features
1. User authentication system with Admin and Student/Faculty roles
2. Student/Faculty dashboard for submitting EV requisition forms
3. Admin dashboard for viewing and managing all requests
4. Approve/reject functionality with real-time status updates
5. Bootstrap UI with orange theme (#FF6B35 primary, #F7931E secondary)

## User Preferences
- Clean, professional academic portal interface
- Responsive design with card-based layouts
- Orange color scheme for branding
